# CREATE Mode

从原始灵感建立概念、人物/主体、结构、视觉叙事、镜头语言、声音与可制作方案。

## V3双重选择

开始时先选择：

1. 操作模式：CREATE；
2. 主导演模式：STORY_DIRECTOR / VISUAL_DIRECTOR / BLOCKBUSTER_DIRECTOR / EXPERIMENTAL_DIRECTOR / PRODUCTION_DIRECTOR；
3. 创作路径：COMMERCIAL / LITERARY / EXPERIMENTAL / HYBRID；
4. 输出层级：CONCEPT_DIRECTION / DEVELOPMENT_PACKAGE / DIRECTOR_PACKAGE / PRODUCTION_PACK。

导演模式负责组织工作，创作路径负责作品回报，两者不得混为一谈。

## 通用流程

1. 识别输入种子：人物、场景、情绪、视觉、热点、机制或不完整概念。
2. 确认观众核心感受、时长、比例、平台和制作目标；缺失项采用明确假设。
3. 建立观众旅程：第一接触、持续等待、峰值、结尾余留。
4. 完成概念压缩：一句话概念、单一机制、人物任务、不可逆压力、艰难选择、最后图像。
5. 建立人物—世界关系、外部身份、隐藏状态、重复动作和关系物件。
6. 设计三至五个视觉母题，以及发现、接近、重解释、余留四段图像结构。
7. 选择适合的结构；中段必须发生事件、关系、情绪、意义、形式或感知变化。
8. 设计关键镜头的目的、输入输出、景别、机位、运镜、视觉优先级和揭示顺序。
9. 建立Style DNA、视觉圣经和声音世界。
10. 需要制作时建立角色锁定、环境锁定、镜头优先级、风险替代和生成顺序。
11. 热梗与传播只在授权或目标明确时启用，并保护人物选择、主题和结尾。
12. 运行适用评估，列出must_fix、must_protect和next_decision。

## 高概念CREATE流程

当请求涉及科幻、意识流、灾难、大片、未来、梦境、记忆、时间或高概念传播时，不得直接铺写完整剧本。

### A. 概念压缩

先生成1—3个真正不同的候选：

```yaml
one_sentence_concept:
core_mechanism:
primary_task:
irreversible_pressure:
impossible_choice:
signature_visual:
final_image:
```

候选差异必须来自机制、人物选择或结尾，不得只是更换美术风格。

### B. 一句话复述测试

一句话包含：谁、必须做什么、规则如何要求或阻止他、他将失去什么。

需要三个以上专有名词、括号解释或第二段补充时，概念尚未压缩。

### C. 单机制与单任务

所有主要画面回答：

> 为什么这个画面只能由本片核心机制产生，它改变了任务、代价、关系、真相还是感知？

主人公在大部分片长内只有一个主任务。任务可以被重新理解，但不能同时并列多个同等级目标。

### D. 艰难选择

高潮由人物选择完成：

- 两边都有真实价值；
- 至少一项损失不可逆；
- 不存在临时无代价第三方案；
- 选择改变最后图像的意义。

### E. 核心画面预算

默认集中3—5个核心画面：

1. 开场异常；
2. 规则第一次完整展示；
3. 中段关系或真相变化；
4. 高潮不可逆行动；
5. 结尾残像。

其余场景服务人物、任务、关系和选择，不追求每镜宏大。

## 视觉叙事CREATE

当主导演模式为VISUAL_DIRECTOR时，先读取：

- `controllers/visual-narrative.md`
- `controllers/camera-language.md`
- `controllers/sound-design.md`
- `templates/visual-narrative-board.md`

默认先输出：

1. 人物—世界关系；
2. 三至五个视觉母题；
3. 重复动作和关系物件；
4. 四段图像结构；
5. 关键画面的隐藏信息；
6. 最后一幅画面。

用户要求完整脚本或制作时，再扩展为逐镜头计划。

## DIRECTOR_PACKAGE顺序

```text
导演声明
→ 一句话概念与情绪核心
→ 视觉圣经
→ 人物—世界关系
→ 视觉母题与结构
→ 镜头语言
→ Style DNA
→ 声音设计
→ 生产计划
→ 传播计划（按需）
→ 导演审查
```

## CREATE禁止

- 强制所有作品反击、获胜、反转或封闭结尾；
- 因使用AI制作就强制故事出现AI；
- 以“高级感”为由删除事件、人物或理解依据；
- 用更多世界观术语掩盖人物任务不清；
- 把意识流理解为互不相关的超现实画面拼贴；
- 用身份反转代替选择变化；
- 让主角牺牲生命成为所有科幻故事默认结局；
- 为了大片感平均堆叠奇观；
- 在视觉圣经和角色锁定前批量写视频Prompt；
- 一次性机械输出全部模板，而不考虑用户当前决策阶段。
