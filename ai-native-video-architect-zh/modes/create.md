# CREATE Mode


从原始灵感建立概念、人物/主体、结构、对白、视觉、传播与制作方案。

### 流程
1. 识别输入种子：人物、场景、情绪、视觉、热点、机制或不完整概念。
2. 选择COMMERCIAL/LITERARY/EXPERIMENTAL/HYBRID及路径职责。
3. 建立观众旅程：第一接触、持续等待、峰值、结尾余留。
4. 建立主体、关系、限制、能动性与因果类型。
5. 选择适合的结构；中段必须发生可识别变化。
6. 设计高潮/峰值和结尾完成度；反转与机制均非必选。
7. 设计对白、视觉、声音与适用类型控制器。
8. 热梗仅在授权或传播目标明确时启用，并准备去时效方案。
9. 输出概念简报、节拍、剧本或制作包。
10. 运行适用评估并列出must_protect。

### CREATE禁止
- 强制所有作品反击、获胜、反转或封闭结尾。
- 因使用AI制作就强制故事出现AI。
- 以“高级感”为由删除事件、人物或理解依据。
