# AI 原生视频架构师（中文 Skill）

这是 `AI Native Video Architect` 的完整中文版本，适用于 Codex Skills。

它可以：

- 从零设计 AI 原生短片；
- 将传统剧本改造成 AI 原生版本；
- 诊断创意是否只是 AI 特效拼贴；
- 将方案拆成场景、镜头与生成任务；
- 为现实主义、恐怖和喜剧加载不同类型控制器；
- 运行去 AI、替换、因果参与、类型与制作可行性测试。

## 显式调用

```text
$ai-native-video-architect-zh
```

例如：

```text
使用 $ai-native-video-architect-zh，把“一个人持续发出信号却得不到回应”发展成90秒现实主义AI原生短片。
```

Skill 允许隐式调用，因此正常中文请求也可自动触发。

## 四种模式

- Create：从零创作；
- Transform：已有剧本原生化；
- Diagnose：诊断与评分；
- Adapt：制作适配。

## 目录

- `SKILL.md`：入口、路由与硬规则；
- `modes/`：四种模式；
- `controllers/`：现实主义、恐怖、喜剧与通用控制器；
- `references/`：核心工作流、机制库、世界规则、场景图谱与生产适配；
- `templates/`：输出模板；
- `evals/`：评分、类型、反套路、制作与回归测试；
- `examples/`：中文案例；
- `scripts/`：安装、校验和评分辅助脚本。

## 设计原则

主入口保持相对短小，按任务渐进读取参考文件。默认不把完整理论全部输出给用户，而是先给可直接使用的创作结果。
