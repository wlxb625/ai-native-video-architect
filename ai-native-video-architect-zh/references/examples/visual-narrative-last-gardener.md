# 视觉叙事示例：《最后的园丁》

> 本示例用于说明结构，不作为固定题材模板。

## 一句话概念

人类离开地球多年后，一个园艺机器人仍每天照料一株不会生长的植物；直到最后一次浇水，观众才发现它保存的不是植物，而是主人留下的生命记忆。

## 主导演模式

`VISUAL_DIRECTOR + EXPERIMENTAL辅助 + PRODUCTION_DIRECTOR辅助`

## 人物—世界关系

- 外部身份：温室园艺机器人；
- 隐藏状态：它不是等待任务结束，而是在延续与主人共同生活的最后程序；
- 重复动作：清晨开灯、测量土壤、浇一小杯水；
- 关系物件：主人留下的裂纹陶杯；
- 世界变化：城市完全荒废，温室仍被机器人维持；
- 人物变化：机器人最终停止维护系统，把最后能源交给植物。

## 视觉母题

### 裂纹陶杯

1. 第一次：机器人用它量水，只像工具；
2. 第二次：杯底投影出一个人曾握住它的手部轮廓；
3. 第三次：机器人关机后，杯中水被植物根部全部吸收。

### 温室灯

1. 第一次：每天准时亮起；
2. 第二次：城市远处所有灯都已熄灭，只有温室亮；
3. 第三次：机器人关机，温室灯熄灭，植物自身发出微弱暖光。

### 擦叶动作

1. 第一次：机械维护；
2. 第二次：机器人在没有灰尘的叶片上仍重复擦拭；
3. 第三次：回忆投影显示主人曾用相同动作触碰机器人的头部。

## 四段图像结构

### Discovery

超远景：死寂城市被沙尘覆盖，唯一亮着的温室像夜色中的小点。

中景：机器人每日开灯、浇水、擦叶。植物始终没有变化。

异常：温室水箱已接近零，但机器人只给这一株植物用水。

### Approach

机器人拆下自己身体中的冷却液管，接入灌溉系统。

裂纹陶杯中短暂出现一个人的手部投影。机器人停顿，但没有对白。

观众开始理解植物和主人有关，却不知道关系性质。

### Reinterpretation

植物根部亮起，温室玻璃上映出过去生活片段：主人不是种植植物，而是在把自己的记忆编码进种子。

此前的浇水被重解释为机器人每天读取一小段共同记忆。

机器人发现最后一段记忆需要消耗其全部能源才能播放并完成生长。

### Residue

机器人把电源线接入花盆，坐在植物旁边，做最后一次擦叶动作。

所有城市与温室灯熄灭。

沉默后，一片新叶自行展开，发出与温室灯相同的暖光。

裂纹陶杯内传来一次很轻的呼吸声。

## 镜头语言示例

```yaml
shot:
  id: 12
  goal: 把“浇水是读取记忆”从猜测变为可见关系
  input_state: 机器人将最后一杯水倒入土壤
  output_state: 观众看见主人手部投影，机器人第一次停止重复动作
  subject: 陶杯、机械手和湿润土壤
  action: 水沿裂纹流下，根部亮起
  shot_size: 特写转中近景
  angle: 平视略低
  movement: 从杯沿缓慢横移到机器人停止的手
  movement_motivation: 先让观众看见异常，再发现异常改变了人物
  visual_priority: 水滴 → 手部投影 → 机器人静止
  reveal_order: 工具漏水 → 人类手影 → 机器人记得这个动作
  emotional_effect: 温柔、不安、失去
  sound_relation: 水声先正常，随后混入极轻的呼吸和布料摩擦
  ai_generation_risk: 精确手部投影与水流同步
  stable_alternative: 分为水滴特写、投影插入镜头、机器人停顿三个镜头
```

## 声音设计

- 世界声：远处风沙、温室风机、金属结构偶尔收缩；
- 人物声：伺服电机、陶杯碰桌、机器人逐渐不稳定的冷却泵；
- 声音母题：呼吸声；
- 第一次：被误认为温室风机；
- 第二次：从陶杯投影中出现；
- 第三次：机器人关机后仍在新叶旁继续一次。

## 制作策略

- 单角色、单主场景，降低一致性风险；
- 城市大全景只做一个S级建立镜头；
- 核心样片优先测试“陶杯水滴 + 手影 + 机器人停顿”；
- 精确手部投影可用后期合成；
- 机器人关机与新叶展开拆成两个镜头；
- 复杂记忆画面使用玻璃反射和短插入，不做长时间全息人物表演。

## 评估要点

- 故事主要通过动作、物件和空间被理解；
- 三个母题均发生变义；
- 视觉高潮不是城市爆炸，而是机器人停止重复动作；
- 最后图像同时完成人物选择、关系回收和声音余留；
- 即使不解释世界年代，观众仍能复述核心关系。
