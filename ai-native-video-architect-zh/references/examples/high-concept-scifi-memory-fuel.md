# 高概念科幻示例：记忆燃料

本示例用于展示概念压缩方法，不是固定剧情，也不要求每部科幻片使用亲情、牺牲或记忆主题。

## 原始过载版本

一个意识方舟中的已故导航员进入集体梦境，寻找造成系统崩溃的零号意识。梦境包含失重城市、倒置海洋、古代战场、医院、黑色太阳、集体未完成的告别和人工神经导航。她发现自己是异常源，最后分解意识拯救方舟。

问题：

- 多个同等级核心机制并存；
- 人物同时寻找异常、调查身份、面对女儿、修复导航和拯救人类；
- 视觉元素可以互相替换，未全部进入因果；
- “自己就是异常源”和“牺牲自己救所有人”容易预判；
- 结尾完成但不够独特。

## 压缩后的核心

```yaml
one_sentence_concept: 一艘星际方舟靠燃烧乘员的记忆航行，最后一次跃迁必须烧掉导航员关于女儿存在过的全部记忆。
core_mechanism: 记忆是推进燃料；越重要的记忆，能量越高。
primary_task: 把被系统选中的记忆锚点送入记忆引擎。
irreversible_pressure: 方舟维生系统将在倒计时结束后停止。
impossible_choice: 保留女儿最后的存在证据，或让四万名乘员抵达新世界。
signature_object: 红色纸船。
final_afterimage: 她抵达新世界，手里仍握着纸船，却不知道它属于谁。
```

## 为什么更强

### 世界规则与人物冲突相同

方舟需要燃料不是背景问题。燃料本身就是人物最不愿失去的记忆。

### 所有奇观来自一个机制

- 被遗忘的人失去重力；
- 城市被拉成光线，因为城市记忆正在进入引擎；
- 女孩先失去声音、再失去脸，因为人物记忆正在被预热删除；
- 倒置海洋来自未兑现的看海承诺；
- 新世界的海洋兑现了承诺，但人物已经忘记承诺对象。

### 人物可以活下来

让人物继续生活但失去最重要的意义，比默认“牺牲生命化为光”更残酷，也更有余震。

### 结尾来自选择

红色纸船不是随机诗意符号。它在开场建立关系，在高潮成为记忆锚点，在结尾成为没有解释但仍存在的身体残余。

## 五个核心节拍

1. **钩子**：深空地铁中，系统宣布必须燃烧女儿的记忆。
2. **规则**：人物穿过正在被燃烧的城市，看见遗忘如何改变物理世界。
3. **私人化**：倒置海洋和儿童房暴露未兑现的承诺。
4. **选择**：记忆引擎前，人物必须确认目标姓名和关系。
5. **残像**：抵达后人物忘记女儿，却本能保留纸船并回应一个不存在的声音。

## HERO_SHOT

```yaml
- role: HOOK_SHOT
  image: 地铁在深空穿过正在卷曲燃烧的地球。
  story_change: 系统宣布最终记忆候选。

- role: RULE_SHOT
  image: 失重城市被拉成发光线条吸入记忆引擎。
  story_change: 人物和观众第一次看懂记忆如何成为燃料。

- role: ESCALATION_SHOT
  image: 倒置海洋穿过儿童房，女孩的声音开始消失。
  story_change: 宇宙规则触及人物私人关系。

- role: CLIMAX_SHOT
  image: 人物亲口确认女儿姓名与关系，红色纸船化为推进光流。
  story_change: 人物完成不可逆选择。

- role: AFTERIMAGE_SHOT
  image: 新世界海面日出，人物握着纸船却不知道来源。
  story_change: 证明成功真实发生，同时暴露成功的代价。
```

## 不应重新塞回的内容

- 主角其实早已死亡；
- 主角就是零号意识；
- 所有人的未完成告别形成集合人格；
- 第二套人工导航牺牲规则；
- 与记忆燃烧无关的战场、医院和古城；
- 结尾由陌生角色继承全部记忆。

这些点子单独可能很好，但放进同一部六分钟短片会分散核心。

## 可迁移方法

将“记忆燃料”替换成其他单机制：

- 飞船以乘员未来寿命作为燃料；
- 城市通过删除居民姓名维持现实稳定；
- 每次时间旅行都会让一个共同记忆从所有人脑中消失；
- 机器人修复身体时必须用一段真实人生替换损坏部分；
- 平行世界的出口只在一个人彻底忘记另一个人后开启。

无论换成什么机制，仍保持：

`一个规则 → 一个任务 → 一个私人代价 → 一个选择 → 一个结尾残余`。
