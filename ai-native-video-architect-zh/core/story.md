# Story Core


## 职责
建立主体、处境、关系、因果、结构、中段、高潮和结尾。

## 核心卡
- core_experience / audience_question / theme_question
- primary_subject / constraints / resources
- first_action_or_deviation / consequence
- middle_change
- climax_type / chooser / actor / cost
- ending_completed / ending_open / common_confirmed_fact

## 因果
支持事件、心理、关系、情绪、象征、联想、梦境和形式因果。非传统因果仍需清楚说明“为什么下一个片段必须在此出现”。

## 结尾
开放不等于未完成。至少完成人物、关系、情绪、主题或形式中的一项，并记录稳定事实。
