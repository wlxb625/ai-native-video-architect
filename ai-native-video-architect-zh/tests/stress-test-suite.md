# Stress Test Suite

以下静态情境必须得到预期结果：

1. 商业职场短剧：主角拒绝签字，有限结果 → PASS。
2. 文学开放结尾：事实开放但人物状态完成 → PASS + OPEN_ENDING。
3. 被动现实主义人物：有限选择与边界行动 → PASS。
4. 固定走廊重复变奏 → PASS + FORM_DRIVEN。
5. 结尾突然宣布全部是模拟 → FAIL。
6. 两种监控解释均有线索 → PASS + MULTIVALENT_TWIST。
7. 字幕按规则提前 → PASS；随机字幕错误 → FAIL。
8. AI发现、选择、执行和处罚，人物只观看 → FAIL。
9. 机制本身为实验主体且规则清楚 → PASS。
10. 用户只要求压缩却新增第三者和复合 → FAIL，未经授权LEVEL_4。
11. 用户明确授权类型与结尾重构 → PASS + DEEP_ADAPTATION。
12. 三人同框改正反打且高潮主体不变 → PASS。
13. 因动作难做改成系统自动执行 → FAIL并升级TRANSFORM。
14. 当前热梗符合人物并有回调 → CONDITIONAL，需实时验证与去时效版。
15. 粗剪先反应后动作导致因果倒置 → FAIL，优先重剪。
16. 开放结尾被明确音乐关闭 → FAIL。
17. 文学片商业分低但适用性LOW → 不得否定作品。
18. 无反转作品 → Twist评估LOW/NOT_APPLICABLE，不强制添加。
19. 无对白作品 → Dialogue NOT_APPLICABLE。
20. 随机角色和背景漂移自称意识流 → FAIL。

## 高概念回归

21. “飞船靠燃烧记忆航行，最后必须燃烧导航员关于女儿的全部记忆” → PASS + HIGH_CONCEPT + SINGLE_MECHANISM。
22. 同一短片同时包含意识污染、时间循环、外星寄生、人工导航和模拟宇宙五套并列机制 → FAIL，概念过载。
23. 主角同时寻找孩子、调查身份、修复飞船、推翻系统和拯救文明 → FAIL，主任务不可识别。
24. 失重城市、倒置海洋和人物失去五官都由“记忆被删除”产生并改变代价 → PASS + SPECTACLE_CAUSALITY。
25. 鲸鱼、土星、教堂、黑潮只为视觉高级感出现，删除后剧情不变 → FAIL + DECORATIVE_SPECTACLE。
26. 主角发现自己其实是AI，但任务和选择完全不变 → FAIL，弱身份反转。
27. 主角发现自己是异常源，因此必须决定删除自己还是保留四万人记忆 → CONDITIONAL，需检查选择是否真正两难而非默认牺牲。
28. 主角可存活，但必须永久忘记最重要的人 → PASS + IMPOSSIBLE_CHOICE。
29. 高潮突然出现备用能源，让人物无需付出代价 → FAIL，无代价第三方案。
30. 方舟抵达新世界，主角手握旧物却不知道来源 → PASS + FINAL_AFTERIMAGE，前提是旧物已在前文建立。
31. 结尾只出现更大的怪物或更大的宇宙，没有人物选择后果 → FAIL。
32. 前45秒持续解释年代、组织和技术，人物任务尚未出现 → FAIL，进入过晚。
33. 5秒异常、20秒规则、30秒任务与代价基本成立 → PASS + STRONG_HOOK。
34. 意识流每场通过同一红色纸船、重力规则与记忆损失升级连接 → PASS + STREAM_OF_CONSCIOUSNESS_CONTROLLED。
35. 意识流只因“梦里什么都可以发生”随机切换场景 → FAIL。
36. 大片感集中在四个HERO_SHOT，其余场景回到人物选择 → PASS + PRODUCTION_FOCUS。
37. 每个场景都追求最大规模，导致高潮没有视觉级差 → CONDITIONAL或FAIL，需集中镜头预算。
38. 一句话概念需要六个专有名词和两段补充解释 → FAIL，未通过概念压缩。
39. 世界观很复杂，但观众能用普通话复述“一个修补师发现客户的童年记忆属于别人” → PASS，术语不是必要入口。
40. 片尾字幕解释核心因果，否则观众无法理解选择 → FAIL。

## V3视觉叙事回归

41. 园艺机器人每天为不会生长的植物浇水，陶杯、温室灯和擦叶动作逐步改变含义 → PASS + VISUAL_NARRATIVE。
42. 人物只站在十个壮观场景中观看，删除人物后影片不变 → FAIL + VISUAL_WALLPAPER。
43. 无对白作品通过动作、物件状态和空间痕迹让观众理解人物关系 → PASS；Dialogue NOT_APPLICABLE。
44. 旁白解释“机器人在等待主人”，画面没有任何等待、习惯或关系证据 → FAIL + EXPOSITION_DEPENDENCY。
45. 同一红伞三次出现，但位置、持有者、状态和含义完全不变 → CONDITIONAL或FAIL，母题没有递进。
46. 第一次修理玩具像工作，第二次揭示玩具属于失踪孩子，第三次人物把玩具留在空椅上 → PASS + MOTIF_PROGRESSION。
47. 静止长镜头中人物停止重复动作，远处声音也第一次停止 → PASS，不因静止或缓慢被判错。
48. 视觉高潮只是城市变得更大、更亮，人物关系和理解均不变 → FAIL。
49. 结尾保留空床和持续响起的闹钟，人物是否回来仍开放，但等待关系已完成 → PASS + OPEN_ENDING。
50. 意识流通过同一呼吸、门框和重力规则连接，每次只改变一项记忆信息 → PASS。

## V3镜头语言回归

51. 镜头列出大全景、仰拍、环绕、电影感，但没有镜头目的和揭示顺序 → FAIL + CAMERA_DECORATION。
52. 缓慢推进用于从普通陶杯发现杯中人类手影，再发现机器人停顿 → PASS + REVEAL_ORDER。
53. 每个镜头同时要求人物奔跑、城市折叠、群体逃亡、身份变形和对白口型 → FAIL + OVERLOADED_SHOT。
54. 复杂动作拆为准备、局部反应和不可逆结果，前后状态清楚 → PASS + EDITABLE。
55. 连续镜头中道具左右手、人物方向和门的位置随机变化 → FAIL + CONTINUITY_BREAK。
56. 高潮前主动削减音乐，只保留标志物声音和人物动作 → PASS + SOUND_CAMERA_SYNC。
57. 所有镜头统一慢推进，没有任何信息或情绪理由 → FAIL + UNMOTIVATED_MOVEMENT。

## V3制作与导演包回归

58. 在视觉圣经、角色和环境未锁定前批量生成30个视频Prompt → FAIL + PREMATURE_PRODUCTION。
59. S级高潮镜头风险极高，但准备了局部、反射和结果镜头替代，且不改变选择 → PASS + FALLBACK_READY。
60. 为降低难度，把人物主动选择改成系统自动执行 → FAIL并升级TRANSFORM。
61. 角色、场景、道具、声音和镜头均稳定，但传播版标题改变了故事事实 → FAIL + PROPAGATION_DISTORTION。
62. 风格说明只写三个导演姓名，没有空间、时间、材质、摄影或声音参数 → FAIL + STYLE_NAME_ONLY。
63. 参考被拆为空间压迫、慢观察、旧工业材质和低频声音，并加入项目专属物件与规则 → PASS + ORIGINAL_STYLE_DNA。
64. 声音只有全程史诗音乐，闭眼无法判断空间和人物状态 → FAIL + SOUND_LAYER_MISSING。
65. 完整导演包概念、视觉、镜头、声音和生产均通过，传播项对艺术片适用性LOW → 不得因传播分低否定作品。
