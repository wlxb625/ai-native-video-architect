# Production Check

Score 0–10:

- character consistency;
- action controllability;
- space continuity;
- prop continuity;
- text/UI accuracy;
- core mechanism feasibility;
- sound feasibility;
- post-production load;
- fallback quality.

## Red flags

- long continuous multi-person interaction;
- exact hand use of small objects;
- repeated face/lip continuity across many scenes;
- changing architecture without fixed references;
- multiple independent objects moving simultaneously;
- precise Chinese text left to video generation;
- a core effect with no minimum viable expression.

## Required output for Adapt

- highest-risk shot;
- stable implementation;
- minimum viable implementation;
- failure fallback;
- final verdict.
