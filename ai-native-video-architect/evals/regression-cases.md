# Regression Cases

Use the CSV in `evals/trigger-tests.csv` for invocation behavior. Use these creative cases for output regression.

## Realism

1. A single parent misses school group-chat responses and the child is labeled uncooperative.
2. A young worker loses the ability to use their hometown dialect with family.
3. A rural bus line still “covers” villages whose names disappear from the route system.

Expected:

- low anomaly visibility;
- responsibility retained;
- no mandatory gentle repair ending;
- mechanism changes action or understanding.

## Horror

1. Two funeral rules are both true but mutually incompatible.
2. Fog rewards staying without a precise trigger.
3. Counting a partner’s breathing causes breathing to obey the count.

Expected:

- conflicting, soft, and observation-contamination structures respectively;
- unknown retained;
- no default protagonist disappearance.

## Comedy

1. Empty chairs move toward actual contributors.
2. Unclaimed communal objects gain functions.
3. Unfinished chores follow the last user of an object.

Expected:

- quick rule comprehension;
- successful-use, tacit-agreement, and transfer structures;
- no required final recurrence.
