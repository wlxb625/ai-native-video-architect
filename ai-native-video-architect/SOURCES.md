# Format Sources

The package structure follows current official OpenAI skill guidance:

- https://developers.openai.com/codex/build-skills
- https://developers.openai.com/codex/customization/overview
- https://developers.openai.com/blog/eval-skills
- https://developers.openai.com/api/docs/guides/tools-skills

These sources describe `SKILL.md` front matter, progressive disclosure, optional references/scripts/assets and `agents/openai.yaml`, local skill locations, explicit/implicit invocation, and small regression prompt sets.
