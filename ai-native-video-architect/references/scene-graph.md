# Scene Graph

Build scene nodes before writing full dialogue or shot prompts.

## Required fields

| Field | Question |
|---|---|
| Time/place | What reality anchor keeps the audience oriented? |
| Reality event | What ordinary event occurs? |
| Character goal | What does the character want now? |
| Trigger/tendency | Why does the abnormal response appear here? |
| Visible/audible change | What exactly can the audience perceive? |
| Character interpretation | What do they currently believe? |
| Choice | What do they do because of this? |
| Visual escalation | What changes in form? |
| Plot escalation | What changes in stakes or options? |
| Theme escalation | What deeper relation is exposed? |
| Genre function | Empathy, suspense, fear, prediction, joke, clue? |
| Audience knowledge | What does the audience understand now? |
| Sound | Trigger, response, silence, synchronization, mismatch? |
| Medium | Generation, sound, editing, compositing, text, practical? |
| Deletion test | What is lost if the node is removed? |

## Scene count guidance

- 30–60 sec: 3–5 structural nodes
- 60–120 sec: 5–8 structural nodes
- 3–5 min: 8–15 structural nodes

Shots may be more numerous than scene nodes.

## Causal scene test

Each major node should change at least one:

- goal;
- relationship;
- understanding;
- rule state;
- audience interpretation;
- risk.

Merge scenes that only show a different-looking instance of the same effect.

## Type additions

### Realism
Add:

- real responsibility;
- type-drift risk;
- action impact;
- understanding impact.

### Horror
Add:

- incorrect explanation;
- unknown retained;
- creator/audience/character knowledge ratio.

### Comedy
Add:

- joke escalation;
- thematic escalation;
- harm level;
- rule comprehension point.
