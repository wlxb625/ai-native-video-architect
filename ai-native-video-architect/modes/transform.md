# Transform Mode

## Use when

The user provides an existing story, screenplay, scene outline, storyboard, or traditional film idea and wants an AI-native conversion or stronger generative expression.

## First protect the source

Explicitly identify:

- characters to preserve;
- central conflict;
- ending;
- genre/reality level;
- required scenes or dialogue;
- elements the user forbids changing.

Do not alter protected elements.

## Diagnose before changing

Locate:

- invisible conflict already present;
- scenes dominated by explanation;
- AI elements that are decorative;
- repeated effects without causality;
- opportunities for environment, sound, object, time, information, or relationship behavior to carry meaning.

## Select transformation strength

### Light
Preserve almost all plot and scenes. Add one motif and low-intensity world responses.

### Medium (default)
Preserve characters, central conflict, and major ending. Rebuild selected scenes around a coherent mechanism.

### Deep
Preserve only theme and central relationship. Use only with explicit permission.

## Default output

- Current-source assessment
- Protected elements
- Main weaknesses
- Transformation strength
- New primary mechanism
- Changed world behavior
- Revised scene structure
- Before/after comparison
- Impact on production

## Quality gate

The revised mechanism must alter character choice, conflict, interpretation, or ending. If it only changes appearance, label the result AI-enhanced rather than AI-native.
