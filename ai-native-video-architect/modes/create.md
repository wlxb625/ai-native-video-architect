# Create Mode

## Use when

The user wants a new concept, short film, narrative video, MV narrative, genre idea, or multiple directions from a seed.

## Procedure

1. Infer project assumptions.
2. Generate three core-proposition candidates internally.
3. Generate 3–5 structurally distinct mechanism candidates internally.
4. Select the strongest unless the user requests multiple directions.
5. Build the rule/tendency and apply the genre controller.
6. Build 5–8 scene nodes for a 60–120 second default.
7. Run pressure tests.
8. Output at requested depth.

## Default Level 2 output

- Working title
- One-line concept
- Type and tone
- Core expression
- One-line AI-native mechanism
- Rule/tendency progression
- Main character(s)
- 5–8 scene nodes
- Ending
- Why it is AI-native
- 2–4 production cautions

## Multiple direction requests

For each direction vary the abnormal source and carrier. Do not produce three cosmetic versions of the same mechanism.

Keep each direction concise, then recommend one and state why.

## Avoid

- writing a conventional plot first and decorating it afterward;
- over-explaining theory;
- giving a complete shot list unless requested;
- using more than one main mechanism by default.
