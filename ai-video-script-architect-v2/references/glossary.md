# Glossary

- **Mode**：CREATE / TRANSFORM / DIAGNOSE / ADAPT。
- **Primary Path**：COMMERCIAL / LITERARY / EXPERIMENTAL / HYBRID。
- **Status**：PASS / CONDITIONAL / FAIL。
- **Design Tags**：描述有意设计，不是失败状态；标签集合可扩展。
- **Applicability**：HIGH / MEDIUM / LOW / NOT_APPLICABLE。
- **BUG / RISK / CHOICE / OPPORTUNITY / PREFERENCE**：诊断五分类。
- **Agency**：主体对作品变化产生不可替代影响的方式。
- **Common Confirmed Fact**：开放解释之间共同成立的稳定事实。
- **Fair Clueing**：反转前已提供理解新答案所需条件。
- **Mechanism**：持续影响事件、人物或观看方式的规则系统。
- **Deletion Test**：删除机制后作品失去什么。
- **Allowed/Forbidden Drift**：生成过程中允许/禁止的变化。
- **Core Sample**：批量生产前验证最高风险的样片。
- **Single Point of Failure**：一项失败会使作品核心不成立。
- **Transform Level**：LEVEL_0校对至LEVEL_4深度改编。
- **Trend Lifecycle**：EMERGING / RISING / PEAK / SATURATED / DECLINING / ARCHIVED / UNKNOWN。
- **Timeless Version**：移除短期热梗后仍保留相同功能的版本。
